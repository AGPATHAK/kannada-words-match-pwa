# candidate_deck_51_100_draft1.md

## Candidate Deck 51–100 — Draft 1

This artifact defines the second curated 50-word set for Milestone 4.

Content decisions locked here:
- schema stays compatible with the current app deck model
- words 51–100 are marked `difficulty_band: "extended"` to distinguish them from the original core 50 while staying high-frequency and everyday-useful
- the set remains mostly noun-based
- only four high-utility verbs are included, all in the final stage
- stages 6–10 continue the staged-introduction pattern with 10 words per stage

## Stage placement philosophy

- Stage 6: school, transport, and carry-item vocabulary
- Stage 7: household, clothing, and daily-use objects
- Stage 8: people and community places
- Stage 9: additional animals, nature, and body-extension nouns
- Stage 10: time words plus a very small set of high-utility verbs

## Notes

- This second 50 is intentionally not a random grab-bag; it expands everyday life coverage around school, transport, household items, community places, and a few essential daily actions.
- The set remains noun-heavy by design.
- The verb count is intentionally low and delayed to Stage 10 so the 51–100 band stays coherent with the original beginner-recognition gameplay.
- No learner-state fields are added to the deck schema.
